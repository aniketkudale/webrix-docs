Get the previous value of a given variable.

## Basic Example

<Example file='hooks/useprevious/BasicExample' height={150}/>

In this example we are passing the entire props object as a value to the hook.
After the initial render, `prev` will be assigned with the props object from the previous render.

## Syntax

```js
const prev = usePrevious(value);
```

### Importing

```jsx
// ESM
import {usePrevious} from 'webrix/hooks';
import usePrevious from 'webrix/hooks/usePrevious'; // For better tree shaking

// UMD
const {usePrevious} = Webrix.hooks;
```

### Return Value

Returns the previous value of the given variable.

> NOTE: in the initial render, the hook will return the **current value** as the previous value.

### Parameters

name|type|description
---|---|---
`value`|any|Some variable/prop
