The `<Zoomable/>` component can be used to to create items that can be zoomed in out programmatically.
It takes a `zoom` prop, specifying the scale factor for the children.

## Basic Example

<Example file='components/zoomable/BasicExample' height={300}/>

In this example you can see how a zoom of `1` will show the content in its actual size, a zoom of `1.5`
will scale the content by 150%, and a zoom of `2` will double the content size.

## Zooming Example

<Example file='components/zoomable/Zooming' height={300}/>

In this example you can use the plus/minus buttons in order to see how the content is zoomed in/out respectively.

## API

### Importing

```jsx
import {Zoomable} from 'webrix';
import Zoomable from 'webrix/Zoomable'; // For better tree shaking
```

### Props

Name|Type|Default|Description
---|---|---|---
`zoom`|number|1|Specifies the zoom level on the scale of 1 where 1 is 100%, 0.5 is 50%, etc.
